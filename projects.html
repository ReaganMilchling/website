Voxel video game

This is the latest video game I've been working on. It is a voxel based renderer written in C++ using OpenGL and CMake. I am trying to keep the number of libraries I use to minimum because I like writing things myself.

Here's a few pictures. It looks suspiciously like Minecraft. I was quite stunned at how easy it was to imitate Minecraft's water look. I clearly haven't figured out how to shade the voxel sides different colors nor implement ambient occlusion.


TDF IDF search engine in C

In College I took a class called Information Retrieval. The class was basically a semester long project on building a search engine. I originally wrote it in (some terrible) Python code.

I wanted to challenge myself so I decided to rewrite it in C. And I took the oppurtunity to write generic hashmaps in C. I also improved single thread parsing time 25x, which was pretty satisfying.
